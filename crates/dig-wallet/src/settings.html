DIG protocol

Settings for how this browser participates in the DIG Network as a native node.

Local cache ● Local-first

Your browser is a DIG node. When you open a dig:// site, it downloads that store's .dig file to a local cache and serves it from your machine — instantly, and verified against the chain. Pages served from this cache show a local badge in the address bar. When the cache is full, the least-recently-visited stores are evicted first (LRU).
