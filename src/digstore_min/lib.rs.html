1//! Digstore Min - A simplified content-addressable storage system
2//!
3//! Digstore Min provides Git-like repository functionality with enhanced merkle proof
4//! capabilities and URN-based retrieval. It focuses on core content-addressable storage
5//! without encryption, privacy features, or blockchain integration.
6//!
7//! # Core Features
8//!
9//! - **Content-Addressable Storage**: Every piece of data identified by SHA-256 hash
10//! - **Layer-Based Architecture**: Data organized into layers similar to Git commits
11//! - **Merkle Proofs**: Generate cryptographic proofs for any data item or byte range
12//! - **URN-Based Retrieval**: Permanent identifiers with byte range support
13//! - **Portable Format**: Self-contained repositories that work anywhere
14//!
15//! # Example Usage
16//!
17//! ```rust,no_run
18//! use digstore_min::{Store, StoreId};
19//! use std::path::Path;
20//!
21//! // Initialize a new repository
22//! let mut store = Store::init(Path::new("./my-project"))?;
23//!
24//! // Add files to the repository
25//! store.add_files(&["src/main.rs", "Cargo.toml"])?;
26//!
27//! // Create a commit
28//! let commit_id = store.commit("Initial commit")?;
29//!
30//! // Retrieve a file
31//! let content = store.get_file(Path::new("src/main.rs"))?;
32//! # Ok::<(), Box<dyn std::error::Error>>(())
33//! ```
